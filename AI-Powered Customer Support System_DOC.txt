  AI-Powered Customer Support System using NLP,BERT & RAG


Project Overview:

This project builds an intelligent AI customer support assistant that can automatically understand customer queries, detect user intent and sentiment, retrieve relevant knowledge, and generate helpful responses.

The system combines machine learning, transformer models, vector search, and generative AI to simulate a real-world customer support chatbot.

The pipeline performs the following tasks:

Understand customer message
Identify the intent of the query
Detect customer sentiment
Retrieve relevant support information
Generate an intelligent response
Deploy as a Streamlit web application.

Vector Database using FAISS :
To provide intelligent answers, the system uses FAISS vector search.

Embedding Model:

SentenceTransformer -- all-MiniLM-L6-v2

Each document is converted into a 384-dimension vector.


Retrieval Augmented Generation (RAG):

RAG Pipeline

User Query
↓
Intent Classification
↓
Sentiment Detection
↓
Vector Search (FAISS)
↓
Retrieve Relevant Document
↓
LLM Generates Response

Generation Model -- GPT-2

The retrieved document is provided as context to the generator.
